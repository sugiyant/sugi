Halo dan selamat datang di blog baru saya!

Blog ini dibuat menggunakan **GitHub Pages** dan **Jekyll**, dengan desain yang bersih dan skema warna yang terinspirasi dari antarmuka GitHub.

### Mengapa GitHub Pages?

GitHub Pages adalah cara yang fantastis untuk menghosting situs web statis langsung dari repositori GitHub Anda. Ini sangat ideal untuk blog pribadi, dokumentasi proyek, atau halaman arahan sederhana.

*   **Gratis:** Hosting gratis untuk situs statis.
*   **Sederhana:** Terintegrasi langsung dengan alur kerja Git Anda.
*   **Cepat:** Situs statis sangat cepat dan aman.

### Fitur Desain

Kami telah menerapkan beberapa penyesuaian CSS untuk memberikan tampilan yang bersih dan profesional, mirip dengan yang Anda lihat di GitHub:

1.  **Skema Warna:** Menggunakan warna biru (`#0969da`) khas GitHub untuk tautan dan elemen penting.
2.  **Tipografi:** Menggunakan font sistem untuk tampilan yang cepat dan akrab.
3.  **Gaya Kode:** Blok kode dan kode inline memiliki latar belakang abu-abu muda dan batas yang jelas, meniru gaya GitHub.
